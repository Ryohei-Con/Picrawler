# Robot Control Project (SunFounder Kit)

## Overview
SunFounder社のロボットキットを用いて、ハードウェア制御と低レベル通信の理解を目的としたプロジェクトです。  
Raspberry Piを中心に、サーボモータ制御およびカメラ画像取得の実装を行いました。

---

## What I Achieved

### 1. I2C通信によるサーボ制御
- サーボモータとRaspberry Pi間の通信にI2Cを使用
- レジスタに書き込む値を理解し、直接制御を実装
- PWM制御の原理から、必要なレジスタ値を逆算して設定

#### 技術ポイント
- PWM信号（パルス幅）とサーボ角度の関係を理解
- I2C通信における「デバイスアドレス」「レジスタアドレス」「データ」の構造を実践的に学習
- 抽象化されたライブラリに頼らず、低レベルで再実装

---

### 2. カメラ画像の取得・保存
- Raspberry Piに接続したカメラから画像を取得
- GStreamerを用いて画像を加工し、ファイルとして保存

#### 技術ポイント
- カメラから取得されるrawデータの扱いを理解
- そのままでは保存できない理由（フォーマット未定義）を把握
- GStreamerによるエンコード処理を利用

※ 現時点では関連スクリプトは未push
